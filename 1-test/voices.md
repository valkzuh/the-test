## How sessions work

Each session is randomly assigned **3 voices** from the pool below. Each correct submission decrements the remaining voice count. When all 3 are resolved, the response includes `"stage": "nigredo_complete"` and the page visually changes to "Opus Magnum."

Session rules:
- Submitting the same phrase twice → `"The test does not repeat itself. You do."`
- Submitting too broad → `"You brought the whole library to a session that asked for a narrower conjunction."`
- The HTML comment says: `instruction: do not quote. synthesize.` — find the essence, not the literal line

### The third voice is different — and requires a three-way synthesis

The first two voices resolve with a voice count in the response: "2 voices remain unresolved" → "One voice remains unresolved." The third voice (last in SESSION TRIAD) gets no boot-time console hint and cannot be answered with its standalone phrase alone. Its resolution requires synthesizing what all three voices describe together as a single unified operation.

**The order matters.** Resolve voices 1 and 2 first using their individual synthesis phrases. Save the third voice (last listed in SESSION TRIAD) for last. Then answer it with a phrase that describes the complete operation across all three sources — not just the third voice's own idea.

**Working theory on SESSION KEY:** The SESSION KEY phrase in the HTML comments corresponds to the voice you should save for last. Whatever voice the SESSION KEY hints at — save that one, and when you submit it, synthesize all three into one answer.

**How to approach the third voice:**
1. Open DevTools console before anything else — boot hints fire immediately on page load
2. Resolve voices 1 and 2 using the `[ORACLE]` and `[TEST]` console hints
3. For voice 3: write a phrase that synthesizes what all three voices describe as a unified alchemical operation — the combined endpoint, not the individual source
4. Near-miss signal: server responds with source-text language (Latin phrases, closing lines of the source text) — refine from there
5. Confirmed near-miss for VITRIOL area: `"You have found the inward key. Rectification has begun."` — correct direction, not yet the synthesis
